# DMProject

Before to execute this project it is necessary to install:

- pip install nltk
- nltk.download('stopwords')
- nltk.download('punkt')
-pip install dash==0.39.0  # The core dash backend
-pip install dash-daq==0.1.0  # DAQ components (newly open-sourced!)
